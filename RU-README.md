# quick-start
[🇬🇧 English](README.md)

Инструкции для быстрого запуска сервера и веб-клиентов Jia с использованием Docker Compose и Ansible.

## Развертывание с помощью Ansible

Этот репозиторий включает Ansible playbook, который координирует полное развертывание Jia. Он автоматически соберет и запустит бэкенд на Go, базу данных PostgreSQL, хранилище Minio S3 и фронтенды на React.

### Предварительные требования
- Установленные Docker и Docker Compose
- Установленный Ansible (`pip install ansible` или `apt install ansible`)

### Запуск скрипта установки

Чтобы развернуть все репозитории за один запуск, выполните следующую команду:

```bash
ansible-playbook deploy.yml -K
```
> **Примечание**: Флаг `-K` (`--ask-become-pass`) обязателен, если вы выберете `yes` для настройки Nginx, так как Ansible потребуются права sudo для установки и настройки Nginx на хост-системе.

Во время выполнения playbook запросит у вас несколько параметров (например, порты, доменное имя, URL-адреса API, пароль базы данных). Затем он:
1. Создаст необходимые файлы `.env` для проектов `server`, `web-admin` и `web-client`.
2. Сгенерирует единый файл `docker-compose.yml`.
3. Соберет и запустит контейнеры Docker.
4. (Опционально) Установит и настроит обратный прокси-сервер Nginx.
